# StreamElements Overlay URL Sharing Tool

A simple, user-friendly web tool for generating and sharing StreamElements overlay URLs. This tool helps streamers easily create shareable overlay links without unnecessary URL parameters.

- **Two Input Methods:**
  1. **Parse Full Link** - Paste a complete StreamElements overlay URL and extract the clean shareable link

- **Clean UI** - Modern, responsive design that works on all devices
- **Instant Feedback** - Real-time validation and error messages
- **No Dependencies** - Pure HTML, CSS, and JavaScript

## ğŸš€ Usage

### Method 1: Parse Full Overlay Link

1. Copy your full StreamElements overlay URL (e.g., `https://streamelements.com/overlay/63f8b2c4d9e1a2b3c4d5e6f7/abc123def456`)
2. Paste it into the first input field
3. Click "Generate Share URL"
4. Click "Copy to Clipboard" to copy the cleaned URL

### Method 2: Build from Overlay ID

1. Find the overlay ID from your StreamElements dashboard (the alphanumeric code at the end of your overlay URL)
2. Paste just the ID (e.g., `abc123def456`) into the second input field
3. Click "Generate Share URL"
4. Click "Copy to Clipboard" to copy the generated URL
